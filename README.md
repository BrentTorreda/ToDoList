# TODO App (.NET + Angular)

### Setup Instructions
1. **Backend**:
   - Navigate to `/TodoAPI`.
   - Run `dotnet restore` and `dotnet run`.
   - The API will be available at `https://localhost:5160` (or check console for port).

2. **Frontend**:
   - Make sure that the port in `src/assets/config.json` matches the backend
   - Navigate to `/todo-ui`.
   - Run `npm install`.
   - Run `ng serve`.
   - Open `http://localhost:4200`.
